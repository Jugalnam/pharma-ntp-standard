## 1. 목적

시스템이 명세된 환경에 올바르게 설치·구성되었음을 입증한다.

## 2. 테스트 케이스

| ID | 검증 항목 | 절차 | 기대 결과 | 결과 |
|----|----------|------|----------|------|
| IQ-001 | Python 버전 | `python --version` | ≥ 3.11 | ☐ |
| IQ-002 | 백엔드 의존성 설치 | `pip install -r requirements.txt` | 오류 없이 완료 | ☐ |
| IQ-003 | Node 버전 | `node --version` | ≥ 20 | ☐ |
| IQ-004 | 프런트 의존성 설치 | `npm install` | 오류 없이 완료 | ☐ |
| IQ-005 | 백엔드 기동 | `uvicorn app.main:app` 후 `GET /api/health` | `200 {"status":"ok"}` | ☐ |
| IQ-006 | DB 연결 | 기동 시 마이그레이션/테이블 생성 | 테이블 생성 확인 | ☐ |
| IQ-007 | NTP 소스 도달성 | 구성된 `source_host`로 NTP 질의 | 응답 수신 | ☐ |
