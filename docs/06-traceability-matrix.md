## 4. 커버리지 점검

- [ ] 모든 URS 항목이 1개 이상의 FS로 전개됨
- [ ] 모든 FS 항목이 1개 이상의 DS로 설계됨
- [ ] 모든 High 위험이 OQ/PQ 테스트로 연결됨
- [ ] 모든 URS 항목이 1개 이상의 OQ/PQ로 검증됨
- [ ] 모든 REG-001 모범사례(BP-n)가 1개 이상의 URS로 매핑됨 (BP-2 이중화는 미할당 — 후속 증분)

> High 위험 RISK-007(인터넷↔폐쇄망 피벗)은 OQ-050으로 연결되며, 배포 네트워크 환경 의존으로 **현장 실행 예정**(개발 증분에서는 DS-040 설계·통제 정의까지 완료).
