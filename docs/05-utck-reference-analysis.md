## 1. 목적

본 시스템은 신뢰 시간 기준으로 **KRISS UTC(k)** 를 채택하고, 공식 프로그램 **UTCk**를 *참조 기준*으로 "인용"한다([URS-001](01-user-requirements.md)). 이 인용이 데이터 무결성의 근거가 되려면, **UTCk가 실제로 어떤 알고리즘으로 동작하는지**를 규명하고 본 시스템의 측정이 그에 **정합(conformance)** 함을 입증해야 한다. 본 문서가 그 규명·정합 근거를 제공한다.

### 2.1 핵심 알고리즘 (NTP/SNTP 오프셋)

UTCk를 포함한 모든 NTP/SNTP 클라이언트는 4개 타임스탬프로 오프셋·지연을 계산한다(RFC 5905 / SNTP RFC 4330).

```
T1 = 클라이언트 송신 시각      T2 = 서버 수신 시각
T3 = 서버 응답 송신 시각       T4 = 클라이언트 수신 시각

오프셋 θ = ((T2 − T1) + (T3 − T4)) / 2
왕복지연 δ = (T4 − T1) − (T3 − T2)
```

UTCk의 동기화는 이 θ만큼 로컬 OS 시계를 보정하는 것이고, 측정/비교는 θ를 표시하는 것이다.

## 3. 본 시스템과의 정합성 (Conformance Basis)

| 관점 | UTCk (참조 기준) | pharma-ntp-standard | 정합 |
|------|------------------|----------------------|------|
| 프로토콜 | SNTP/NTP | NTP v3 (`ntplib`) | ✅ 동일 계열 |
| 기준 소스 | UTC(KRISS) `time.kriss.re.kr` | 동일 (`source_host` 기본값) | ✅ 동일 |
| 오프셋 산식 | θ = ((T2−T1)+(T3−T4))/2 | `ntplib` 동일 산식(`resp.offset`) | ✅ 동일 |
| 샘플링 | SNTP 단일 샘플 | 다중 샘플 **중앙값**(기본 3) | ✅ 동등 이상([RISK-004](04-risk-assessment.md) 완화) |
| stratum | 서버 stratum 수신 | `resp.stratum` 수신·검증 | ✅ 동일 |
| **역할** | OS 시계를 **보정(step)** | 오프셋을 **모니터링·경고** | ⚠️ **의도된 차이** |

**결론:** 본 시스템은 UTCk와 **동일한 프로토콜·동일한 기준 소스·동일한 오프셋 산식**으로 시각 편차를 측정한다. 차이는 "시계를 고치느냐(UTCk) / 편차를 감시·경고하느냐(본 시스템)"의 **역할**뿐이며, 이는 GxP 모니터링·검증 도구로서 의도된 설계다. 따라서 본 시스템은 **"공식 UTCk에 *준하는*(conformant-to) 측정 기준"** 을 사용한다고 입증할 수 있다.

> 실측 확인(2026-06-20): `time.kriss.re.kr` 질의 성공 — offset ≈ −1.35ms, stratum 3. 기준 소스 도달성([IQ-007](07-iq-protocol.md)) 및 측정 가능성 확인됨.

## 4. "UTCk에 준함"의 입증 절차

본 규명을 실제 검증으로 닫는 두 단계는 이미 프로토콜에 정의되어 있다.

- **[OQ §3](08-oq-protocol.md)** — 수집된 시스템 기준 시각을 UTCk가 표시하는 KRISS 시각과 비교(차이 ≤ 표준 허용 한계).
- **[PQ §3](09-pq-protocol.md)** — 실 운영에서 본 시스템 보고 시각과 공식 UTCk 표시 시각을 **동시 측정·비교**하여 정합 입증.

## 5. 한계 및 주의

- 본 프로젝트는 **공식 인증 제품이 아니며**, UTCk를 *참조 기준*으로 인용한 오픈소스 구현이다.
- `UTCk.exe`는 **타사(KRISS) 바이너리**로 저장소에 포함하지 않는다(`.gitignore`). 교차점검은 로컬 설치본을 사용한다.
- 측정은 네트워크 도달성에 의존한다([RISK-003](04-risk-assessment.md): 폴링 중단 감지로 완화).
- UTCk 4의 SNTP 세부(샘플 수·필터·재시도)는 공개 사양이 제한적이므로, 정합 입증은 **산식·소스·프로토콜의 동일성**과 **교차점검 실측 결과**로 갈음한다.
